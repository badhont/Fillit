# Fillit
  Program that place tetriminos in the most compact configuration

## Built with
  C language

## Installing
  - Git clone the repository
  - Compile it with `make`

## Make it work
  - Find a test file in the `test/` directory
  - Run `./fillit [testfile]`
  - Make your own test file containing tetriminos<br/>
  exemple :
  
```
....
#...
##..
#...

....
..##
.##.
....
```
